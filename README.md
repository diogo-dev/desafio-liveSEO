# Explicação das minhas escolhas

Nos desafio 2, eu escolhi utilizar o `Vue.js`, pois foi a oportunidade de aprender um framework diferente do que estou acostumado (`Next.js/React`). E parece ser o framework que vocês utilizam na liveSEO.

No desafio 5, eu escolhi o `NestJS`, pois tenho familiaridade e, particularmente, gosto bastante do framework.

Cada desafio está dentro de uma pasta numerada e, dentro de cada pasta, tem um arquivo `README.md` com a explicação do código e as escolhas que fiz para implementar o desafio. 

Nos desafios que não tem implementação, o arquivo `README.md` possui a resposta para a questão proposta.
